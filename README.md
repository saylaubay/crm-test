## CRM test

Requirements
- SERVER: Apache 2.4
- PHP: 8.1 or higher
- For MySQL users: 5.7 or higher.

## Quick Start:
$ git clone https://github.com/saylaubay/crm-test.git \
$ cp .env.example .env \
$ composer install \
$ php artisan key:generate \
$ php artisan migrate --seed
